In this post ,lets focus on HTML.We will learn what HTML is all about,and why you should learn it

What is HTML?

HTML stands for Hypertext Markup Language. Its a markup language that web developers use to structure and describe the content of the webpage (not a programming language)



HTML consists of elements that describe different types of contents:paragraphs,links,headings,images,video,etc.Web browsers understand HTML and render HTML code as websites.

In HTML,each element is made up of 3 parts:
  1. The opening tag
  2. The closing tag
  3. the actual element
